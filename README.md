# Doorbell View

Lightweight Home Assistant doorbell viewer and notifications demo project.

Overview
- Simple viewer + alert scripts for Home Assistant doorbell events.
- Includes architecture notes, example scripts, and deployment hints.

Quick start
1. Clone this repository.
2. Review the architecture in `architecture.md`.
3. Inspect `scripts/` for example integration scripts.

Local testing
- This repo contains shell scripts (`scripts/`) and helpers to simulate alerts.
- Tailor the scripts to your Home Assistant setup before deploying.

What to look for (for reviewers)
- `architecture.md` — design decisions and system overview.
- `scripts/` — integration examples and helpers.
